## ⚠️ متطلبات مهمة:

1. **الصيغة:** MP4 فقط
2. **الكودك:** H.264
3. **الصوت:** AAC (اختياري)
4. **الدقة:** 720p أو أعلى
5. **المدة:** 5-15 ثانية لكل إشارة

## 🚀 بعد إضافة الفيديوهات:

1. أعد تشغيل التطبيق: `npm run dev`
2. افتح `/learning/dictionary`
3. اضغط على أي فيديو - سيعمل مباشرة! 🎉
